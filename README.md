# 🤖 ChatGPT Clone

Projeto desenvolvido com foco no uso prático de Inteligência Artificial no desenvolvimento.

## 🚀 Sobre o projeto

Este projeto simula um chat com IA, inspirado no ChatGPT, permitindo o envio de mensagens e recebimento de respostas dinâmicas.

O principal objetivo não foi apenas construir a aplicação, mas aprender a utilizar ferramentas de IA de forma profissional no contexto de desenvolvimento.

## 🎯 Objetivo

- Aprender a usar IA como assistente de programação
- Criar prompts eficientes
- Aplicar IA na resolução de problemas
- Entender como integrar IA em aplicações reais

## 🛠️ Tecnologias utilizadas

- React (frontend)
- Node.js + Express (backend)
- API de Inteligência Artificial
- CSS

## 💻 Funcionalidades

- Envio de mensagens
- Respostas geradas por IA
- Interface estilo chat

## 📂 Estrutura do projeto

/web → frontend  
/server → backend  

## ▶️ Como rodar

### Backend
cd server  
npm install  
npm run dev  

### Frontend
cd web  
npm install  
npm start  

## 📚 Formação

Baseado na formação **ChatGPT for Devs**, com foco em:
